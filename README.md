# Characters 3D visualizer and clip demonstration


## Introduction
This is simple, demonstration project (not part of any commercial software)
can loaded some english letters (from 2D cointours), creates 3D loft
from 2D contour and apply clipping plane (can be controlled in the application
user interface).

## Prerequisites

Application was developed under Windows 11, with QT 5.12.2 and
compiled using MSVC Community 2022.
